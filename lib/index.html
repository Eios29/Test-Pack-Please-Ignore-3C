This is an MCUpdater serverpack directory. If you're reading this, then Baughn didn't bother to write a proper webpage.

Important: Before running Minecraft, you must configure MCUpdater to allocate more than the default 1G of memory. In the settings screen, change max memory size to 4G or more.
